WIZARDS (Assistentes)

    O Access conta com diversos assistentes (wizards) e também suplementos (add-ins) que realmente fazem muito pelo programador. Alguns já acompanham o Access, como o assistente para Tabelas, Consultas, Formulários, Relatórios, Macros, somente para Módulos não vem com assistente. Ainda existem outros assistentes além dos para Objetos, em Ferramentas - Suplementos temos o Divisor de bancos de dados, Gerenciador de menu de controle. Aí em Ferramentas - Suplementos podemos acrescentas diversos outros Suplementos oferecidos gratuitamente no site da Microsoft ou mesmo desenvolvidos por terceiros e até mesmo por nós (clique aqui e experimente).

    ALGUNS SUPLEMENTOS ÚTEIS:

- Print Relationships - Este imprime um relatório com o gráfico da janela de Relacionamentos
  Ele pode ser conseguido gratuitamente no site de downloads da Microsoft:
  http://www.microsoft.com/downloads/search.asp?
  Selecione sua versão do windows e do Access e procure pelo arquivo PrintRelation80.exe no caso do Access97 ou simplesmente procure por arquivos do Access.

- Command Bar Wizard - Este, como o nome indica, é um suplemento para construção de Barras de Ferramentas. Também é oferecido pela Microsoft em sua seção de Downloads. Procure o arquivo wzcmbr80.exe para o Access97 ou outro para sua versão.

- Wizard Build Wizard - Ajuda a construir Wizards.  Procure na Microsoft o arquivo wzbld97.exe.

- Menu Builder - Este é muito útil e semelhante ao Construtor de menus do VB. Com ele podemos construir menus tipo os do próprio Access e de outros aplicativos windows. Uma grande vantagem dele em relação aos Formulários de menus utilizados com frequência é que ficam bem mais práticos e leves. Peque o arquivo WzMenuBilder.exe no site da Microsoft e instale.
    Vejamos algumas dicas para construir um menu com este Wizard:
    - Abra o assistente
    - Clique em New
    - Selecione New Blak Menu e clique em Next
    - Este é a tela de construção:
            - Em Caption digite o Título do Menu (Ex.: Cadastros)
            - No botão com os 3 pontinhos ao lado de Icon, clique e selecione o Ícone
            - Em ToolTip digite uma dica para o menu que aparecerá no movimento do mouse.
            - Em Action Type clique e selecione RunMacro
            - Em arguments digite o nome da macro
            - Podemos Criar um grupo de macros e digitar assim: NomeGrupo.NomeMacro.

DICA: Ele tem um pequeno bug. Ao levar um sistema com menu para outro micro, mesmo que instalemos também aí neste micro o Construtor de menus, não poderemos editar o menu. Para corrigir isso teremos que levar o arquivo "WzMenu80.mde" do micro onde construimos o menu para o outro micro. Este arquivo está no diretório:
c:\arquivos de programas\microsoft office\office.
