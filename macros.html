MACROS

    Macros são rotinas de programação empacotadas em Ações, de maneira a não exigir de iniciantes e poupar trabalho de experts.

    Com macros podemos fazer muito do que apenas se faria com código e bem mais rápido.

    Claro que as macros tem suas limitações e quando isso acontece temos que partir para a programação via código em Módulos.

Como criar uma Macro para funcionar com Tecla de Atalho?

Basta criar a macro com as ações que deseja para a tecla de atalho e salvar a macro com o seguinte nome:

^S - para a tecla de atalho Ctrl+S
^T - para Ctrl + T e assim por diante.

Criando um Grupo de Macros

Quando precisamos criar várias macros, por exemplo, ao criar um menu com o construtor de Menus (Suplemento encontrado no site da Microsoft - Menu Builder), precisamos criar uma macro para abrir cada um dos Formulários por exemplo.
Fica mais prático criar um grupo de macros, que conterá uma macro em cada linha e será salvo como se fosse apenas uma macro.

- Clique na aba Macros - Novo
- Exiba Nomes de macros, caso ainda não esteja exibido (Exibir - Nomes de macros)
- Digite o nome da primeira (FormClientes) na primeira linha da coluna Nome de macro
- Selecione "AbrirFormulário" na coluna "Ação" e informe abaixo o nome do Formulário "Clientes".
- Procedimento semelhante para tantos quantos sejam os formulários a abrir.
- Feche e salve com o nome do Grupo de Macro "AbrirForm" .

Para fazer referência a cada uma das macros:

=AbrirForm.FormClientes

Veja que nos eventos agora aparecem estas macros.
