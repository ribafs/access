SQL

    SQL -  Structure Query Language - Linguagem estruturada de consultas.

    Esta é uma ferramenta muito eficiente para quem já está avançando em Access, pois sua sintaxe é bastante poderosa, resumida (comparando com o DAO) e eficiente, além de apresentar ganho de desempenho.

Banco de Dados contendo as Consultas SQL do Tutorial abaixo

Veja o tutorial abaixo realmente aplicado em Consultas Access. Com estas Consultas em mãos podemos Criar Formulários, Relatórios, Pesquisas em Campos do formulário (Origem da linha de ComboBoxes), etc.

Pequeno e Prático Tutorial de SQL em Exemplos

O Ajuda do Access é realmente espetacular, mas quando trata do DAO (Recordsets, TableDefs e ) como também da SQL, aí parece que não existe a intençao de descomplicar. Então c/om o intuito de desmistificar esta útil e poderosa linguagem eu adaptei uns exemplos que baixam o nível para melhor compreensão.


    Podemos utilizar SQL também em Recordsets, com rotinas contidas em strings.

    Dim r As Recordset
    r = CurrentDb.Execute("SELECT * FROM tblClientes;")

    Com estas duas linhas apenas, criamos uma consulta que seleciona todos os registros "*" da Tabela tblClientes.
