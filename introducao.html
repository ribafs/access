INTRODUÇÃO

    O MS Access97 é um Sistema Gerenciador de Bancos de Dados Relacional.

    SGBDR é um programa criado para gerenciar bancos de dados relacionais.

    Bancos de dados relacionais são aqueles que podem interligar tabelas pelos seus campos chaves.

    Bancos de Dados são programas utilizados para ajudar a organizar informações, armazenar e recuperar de maneira rápida e eficiente.

    O segredo do aprendizado do Access começa na instalação do mesmo. Temos que realizar uma instalação completa ou então bem personalizada, colocando todos os componentes necessários a construção dos bancos de dados, como os construtores, bancos de dados de exemplo e cia.

    Para quem realmente tem a pretensão de aprender Access ele já vem praticamente com tudo o que é necessário: O próprio Access, O Help, Os Exemplos e Os modelos.

    O Access em si já ajuda e muito na construção dos Bancos de Dados, não conheço outra ferramenta igual.

    O Help é também expetacular, principalmente a guia Localizar, onde podemos digitar uma frase e ele traz o assunto, com exemplos e tudo o mais.

    Os exemplos que acompanham o Access tem muitas informações úteis e avançadas. São normalmente encontrados em:

c:\arquivos de programas\microsoft office\office\exemplos

São 3: soluções, nortwind e pedidos. A microsoft em seu site oferece outros.

Os modelos são uma complementação dos exemplos. Crie um banco de dados com o assistente de banco de dados, selecione Pedidos, Receitas ou quanquer outro. Depois examine o banco de dados, siga seus eventos, examina a estrutura, observe os códigos responsáveis pelos eventos, personalize e mexa até ter segurança de conseguir reproduzi-los.
 

Veja a seção Curso Online para detalhes sobre bancos de dados e como criá-los.

Claro, que quem quer aprender não está dispensado de passear pela Internet, em busca dos bons sites, do site da Microsoft e outros. Veja ao final do índice da Cantinho dos Amantes do Access, o Microsoft Access WebRing, que já conta com mais de 200 sites sobre Access.
