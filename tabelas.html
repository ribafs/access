TABELAS

    Tabelas - Os dados de um Banco de Dados são armazenados nas Tabelas. Aqui ficam armazenadas as informações e não em Consultas ou Formulários.

    Tabelas bem feitas garantem pelo menos uma boa base ao sistema.

- Temos que ficar atentos para criar cada Tabela contendo dados sobre um único assunto.
- Atentar também para que não armazene campos calculados. Se estes podem ser calculados no momento que precisamos, não justifica que fiquem armazenados, além de corrermos o risco de provocar erros com isso, pois podemos armazenar valores que foram calculados há algum tempo e seus fatores terem sido atualizados. Raramente se justifica armazenar campos calculados.
- Ao criar Tabelas atente para suas propriedades, implementando Máscara, Regras de Validação, Formatos, etc.

Veja mais detalhes sobre Tabelas na seção Curso Online.
