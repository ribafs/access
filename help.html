Help

    O Help do Access é um grande auxiliar do programador que pretende aprender a programar em Access, realmente uma ótima ferramenta.

    Como outros helps ele é dividido em 3 telas: Conteúdo, Íncide e Localizar. Todas as 3 são muito boas, eu gostaria de destacar a Localizar, onde podemos encontrar até frases inteiras como explicações para mensagens de erro e outras. Veja também que sempre que possível temos um ou mais exemplos para um tópico.

Convertendo Bancos de Dados Anteriores para o 2000
A Microsoft Oferece um arquivo no formato Word com detalhes sobre a conversão de bancos de dados de versões anteriores para o Access 2000.

    Programar e desenvolver sistemas não é difícil. O difícil é quando estamos trabalhando em um projeto para atender a uma necessidade particilar e nos deparamos com algo que nunca haviamos visto e não temos de pronto a resposta. Aí sim é que dá um pouquinho de trabalho e onde também temos que fazer trabalho de detetive para descascar o pepino, descobrir onde está o bug ou encontrar uma alternativa de resolver a coisa da melhor maneira. Resolver as coisas também sempre é relativamente fácil, o trabalhoso é resolver da maneira mais prática para o usuário, da maneira mais eficiente e técnica, aí sim. Nestas horas, termos por perto conjuntos de Samples (como os oferecidos pela MS), os Exemplos do Cantinho :), os modelos do Access, os exemplos Soluções, Pedidos e Northwind, termos o Help do Access, termos os Knowledge Bases, termos a colaboração de amigos em boas listas, com certeza irá ajudar a encontrar a solução de maneira mais rápida do que apenas tentando sozinho.

    A Microsoft oferece gratuitamente em seu site um "suplemento" para o Help do Access: os Knowledge Bases. É um arquivo no formado de help (HLP) do windows, com as mesmas características do help do Access, com How Tos sobre os mais diversos tópicos do Access: Formulários, Tabelas, Consultas, Módulos, etc. Ensinam como fazer determinadas tarefas, com dicas passo-a-passo em inglês.
